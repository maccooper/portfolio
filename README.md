## maccooper.dev

This site is build out as a mock of a service. Pulling the repo and running any commands displayed in the terminal should function as displayed on the site.

initially prototyped with vercel and claude. The design landed, but the stack did not. Running react and a node build pipeline to serve a static page made no sense for this project.

rebuilt in vanilla html/css/js

I feel that embracing the new layer of abstraction is important, but so is beign willing to dive in, cleand and refactor code it has produced.

Failure to embrace these layers pragmatically will lead to deteriation of any code you touch.
